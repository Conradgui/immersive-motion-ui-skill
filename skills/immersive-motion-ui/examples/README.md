# Core Examples

Load examples only after Core has classified the task and selected a visual direction. Examples demonstrate transferable decisions; they are not templates or default visual identities.

## Offline Baseline

`minimal-showcases/` contains three fixture-locked before/after pairs:

- `product-operations`: operational hierarchy, exception queue, drawer continuity, and auditable action.
- `brand-adaptive-reuse`: subject-specific evidence, material narrative, and adaptive media composition.
- `commerce-hotel-booking`: comparison, terms, running total, selection continuity, and booking-state disclosure.

These pairs require no framework, CDN, remote font, or network image. Open the HTML directly or inspect the paired source and shared `fixture.js`.

## Network-Enhanced References

`finesse-network/` preserves five authorized upstream Finesse HTML references and its MIT license. Read `finesse-network/SOURCE.md` before use. Some references require a network; others also have intentionally unbundled runtime or image dependencies and are source references rather than runnable demos.

Do not copy an example's brand, palette, page composition, or dependency choice by default. Extract the relevant mechanism, adapt it through the current subject and stack, and retain the Core fallback and verification contracts.
