# Showcase Casebook Tests

Run from the Core repository root:

```bash
node tests/showcase-casebook/run-tests.mjs
```

The suite checks the six paired-case contracts, shared fixtures, causal deltas, honest evidence states, anti-copy guards, routing, and M03 activation.

It does not claim that the textual cases have passed browser, responsive, motion, accessibility, performance, or independent-model evaluation. Those remain separate evidence surfaces.
